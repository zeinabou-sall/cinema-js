 Cinema JS - Application de films

Application web développée en JavaScript permettant de rechercher et afficher des films via l’API OMDb.

 Objectif
Créer une application pour un cinéma permettant aux utilisateurs :
- de consulter les films tendances
- de rechercher des films en temps réel
- de voir les détails d’un film

 Fonctionnalités
Page d’accueil (index.html)
- Affichage de films tendances (API OMDb)
- Poster + titre du film
- Lien vers la page détail
- Bouton "charger plus"

Page recherche (search.html)
- Barre de recherche dynamique
- Résultats en temps réel (sans rechargement)
- Affichage des films (poster + titre)
- Bouton "charger plus"

 Page film (movie.html)
- Titre du film
- Poster
- Description complète
- Genre
- Acteurs

 Technologies utilisées
- HTML5
- CSS3
- JavaScript
- API OMDb
